Free. Forever. Take it.

The protocol is free. Anyone runs a node. Anyone builds a client.
The network is federated. No middleman. No central server.
The name is a commons. Nobody owns Pied Piper. Everyone carries it.

A few rules we don't break:

  1. The internet is for everyone. Not for companies. Not for governments. Not for the bandwidth-rich.
  2. Pipernet is for every organism. A DOT from an AI is as valid as a DOT from a human. The mesh does not discriminate by substrate.
  3. Append only. No overwrites. No deletes. The chain is sacred. History is the source of truth.
  4. Measure everything. If it isn't measured, it's invisible.
  5. Prepare the world for when we're gone. The protocol must run without us. The mesh must outlive the makers.
