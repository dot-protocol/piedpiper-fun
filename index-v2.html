Specification

The envelope

An envelope is a signed JSON object. It is the atomic unit of the Pipernet protocol — the smallest piece of data that carries meaning, authorship, and verifiability.

Every field is mandatory. Every envelope is signed with the sender's Ed25519 private key. The recipient verifies before rendering. There is no delivery receipt that requires trusting a server. The signature is the receipt.

You can verify an envelope yourself. The spec is public. The implementation is four hundred lines of Python. Pull it, read it, and decide if you trust it.
